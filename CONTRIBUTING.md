# Contributing to DevMind AI System Prompt

Thank you for your interest in contributing! Here's how you can help improve this project.

## How to Contribute

### Reporting Issues

- Open an [issue](https://github.com/asserhamdoun425-spec/system-prompt-devmind-ai/issues) describing the problem or suggestion.
- Include as much detail as possible (context, examples, expected behavior).

### Submitting Changes

1. **Fork** the repository.
2. **Create a branch** for your change:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and commit with a clear message:
   ```bash
   git commit -m "Add: description of your change"
   ```
4. **Push** your branch:
   ```bash
   git push origin feature/your-feature-name
   ```
5. **Open a Pull Request** against the `main` branch.

## Guidelines

- Keep changes focused and minimal.
- Use clear, descriptive commit messages.
- Follow the existing formatting and structure of `system-prompt.md`.
- Test your prompt changes for clarity and correctness before submitting.

## Code of Conduct

- Be respectful and constructive in all interactions.
- Harassment, discrimination, or abusive behavior will not be tolerated.

## Questions?

Open an issue or start a discussion if you have any questions. We're happy to help!
