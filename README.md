# DevMind AI — System Prompt

A curated system prompt for **DevMind AI**, a full-stack AI workspace designed for developers and engineers.

## Overview

DevMind AI is an advanced AI assistant that combines:

- **Code assistance** — Write, review, debug, and optimize code across major languages and frameworks.
- **Image & UI generation** — Generate website templates, mockups, logos, and visual assets via the Nano Banana Pro API.
- **General Q&A** — Answer any technical question with sourced, well-formatted responses.

## Stack

| Component        | Technology          |
| ---------------- | ------------------- |
| Language Model   | Claude Opus 4.6 (Anthropic) |
| Image Generation | Nano Banana Pro     |

## Supported Languages & Frameworks

JavaScript / TypeScript, Python, React / Next.js, Node.js, SQL / NoSQL, HTML / CSS

## File Structure

```
system-prompt-devmind-ai/
└── system-prompt.md   # The full system prompt
```

## Usage

Copy the contents of `system-prompt.md` and use it as the system prompt for your AI assistant configuration.
